# CVCraft

A modern CV Builder built with Next.js, React, Tailwind CSS and TypeScript.

## Features

- Multi-step CV Builder
- Live CV Preview
- Professional CV Template
- PDF Export / Print
- Responsive Design
- TypeScript
- Tailwind CSS

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- react-to-print

## Installation

```bash
git clone https://github.com/Rightroc/cvmint.git
cd cvmint
npm install
npm run dev
```

## Folder Structure

```
app/
components/
data/
types/
public/
```
